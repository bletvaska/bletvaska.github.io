## 1. Blink
## Schéma zapojenia
## Blink (kód) ```cpp void setup() { pinMode(LED_BUILTIN, OUTPUT); } void loop() { digitalWrite(LED_BUILTIN, HIGH); delay(2000); digitalWrite(LED_BUILTIN, LOW); delay(500); } ```
## Použité funkcie * [`pinMode(pin, mode)`](https://www.arduino.cc/reference/en/language/functions/digital-io/pinmode/) - Configures the specified pin to behave either as an input or an output. * [`digitalWrite(pin, value)`](https://www.arduino.cc/reference/en/language/functions/digital-io/digitalwrite/) - Write a `HIGH` or a `LOW` value to a digital pin. * [`delay(ms)`](https://www.arduino.cc/reference/en/language/functions/time/delay/) - Pauses the program for the amount of time (in milliseconds) specified as parameter.
## Zapojenie diód ![Zapojenie so spoločnou anódou](images/arduino.multi.function.shield-leds.png) * so spoločnou anódou * rozsvietenie diódy pripojením zeme * zapísaním úrovne `LOW` na príslušný PIN * podobne sú zapojené aj tlačítka
## Blink (aktualizovaný kód) ```cpp #define ON 0 #define OFF 1 void setup() { pinMode(LED_BUILTIN, OUTPUT); } void loop() { digitalWrite(LED_BUILTIN, ON); delay(2000); digitalWrite(LED_BUILTIN, OFF); delay(500); } ```
## Štruktúra programu ```cpp void setup(){ // inicializačná funkcia // spustí sa len raz } void loop(){ // hlavná slučka // spúšťa sa stále dookola } ```
## Úloha: Semafor Vytvorte jednoduchú implementáciu štandardného cyklu semaforu, teda: 1. Červená (5s) 2. Červená + Oranžová (2s) 3. Zelená (blikajúca zelená) (5s) 4. Oranžová (2s) 5. Červená
## 2. Digitálny vstup
```cpp #define ON 0 #define OFF 1 void setup() { pinMode(A1, INPUT); pinMode(LED_BUILTIN, OUTPUT); } void loop() { if(digitalRead(A1) == ON){ digitalWrite(LED_BUILTIN, ON); }else{ digitalWrite(LED_BUILTIN, OFF); } } ```
## Použité funkcie * [`digitalRead(pin)`](https://www.arduino.cc/reference/en/language/functions/digital-io/digitalread/) - Reads the value from a specified digital pin, either `HIGH` or `LOW`.
## Úloha: Prepínanie režimov semafora Rozšírte semafor o nočný režim (blikajúca oranžová). Prepnutie z jedného režimu do druhého realizujte stlačením tlačidla.
## Prerušenie * interrupt * metóda pre asynchrónnu obsluhu udalosti, kedy procesor preruší vykonávanie postupnosti inštrukcií, vykoná obsluhu prerušenia a potom pokračuje v predchádzajúcej činnosti * len na špeciálnych PIN-och (napr. UNO - _2_ a _3_)
## Príklad obsluhy prerušenia ```cpp volatile byte state = LOW; void setup() { pinMode(LED_BUILTIN, OUTPUT); pinMode(interruptPin, INPUT_PULLUP); attachInterrupt(digitalPinToInterrupt(2), blink, CHANGE); } void loop() { digitalWrite(LED_BUILTIN, state); } void blink() { state = !state; } ```
## 3. Sériová linka
## Odosielanie ```cpp void setup() { Serial.begin(9600); } void loop() { Serial.println("Hello world"); } ```
## Prijímanie ```cpp void setup() { Serial.begin(9600); while (!Serial) { ; // waiting for serial port } Serial.print("Enter your name: "); } void loop() { if (Serial.available() > 0) { String text = Serial.readString(); Serial.println("Hello, " + text); } } ```
## DS1820 ![DS1820](images/ds1820.png)
* teplotný senzor * lacný (a populárny) * pracuje na zbernici 1-wire * pre jeho použitie s _Arduinom_ je potrebné použiť samostatné knižnice - [OneWire](https://www.pjrc.com/teensy/td_libs_OneWire.html) a [DallasTemperature](https://github.com/milesburton/Arduino-Temperature-Control-Library) * [datasheet](https://datasheets.maximintegrated.com/en/ds/DS18S20.pdf)
```cpp #include <OneWire.h> #include <DallasTemperature.h> OneWire oneWire(A4); DallasTemperature sensors(&oneWire); void setup(void) { Serial.begin(9600); sensors.begin(); } void loop(void){ sensors.requestTemperatures(); Serial.print("Temperature is: "); Serial.println(sensors.getTempCByIndex(0)); delay(1000); } ```
