## JavaScript 101 ### The Objects Created by mirek
## What is an Object? * An **object** is a collection of **properties**. * A **property** is an association between a **name** (or key) and a **value**.
## Object Creation * with **object constructor** ```javascript const obj = new Object(); ``` * with **object literal** ```javascript const obj = {}; ``` * with **inbuild method** ```javascript const obj = Object.create(); ``` * with **constructor function** ```javascript const obj = new Person(); ```
## Object Literal ```javascript let person = { name: 'Sherlock', surname: 'Holmes', getName: function(){ return `${this.name} ${this.surname}`; } }; // usage console.log(person.name); person['surname'] = 'holmes'; ```
## Constructor Function ```javascript // definition function Person(name, surname, sex = 'M'){ this.name = name; this.surname = surname; this.sex = sex; this.getName = function(){ return `${this.name} ${this.surname}`; }; } // usage let person = new Person('Sherlock', 'Holmes'); ```

## JavaScript 101 ### Classes
## Features * available from ECMAScript 6 (2015) * more convenient syntax to create old-school constructor functions * support for class fields and instance fields * subclassing (class inheritance)
## Class Example ```javascript class Account{ constructor(balance = 0){ this.balance = balance; } withdraw(amount){ this.balance -= amount; } deposit(amount){ this.balance += amount; } get_balance(){ return this.balance; } } ```
## Class Instantiation ```javascript const account = new Account(); account.get_balance(); // 0 const account2 = new Account(100); account2.get_balance(); // 100 ```

## Subclassing ```javascript class SavingsAccount extends Account { constructor(balance = 0, overdraftLimit = 0) { super(balance); this.overdraftLimit = overdraftLimit; } withdraw(amount){ if(this.balance - amount < this.overdraftLimit){ throw 'Insufficient Funds'; }else{ super.withdraw(amount); } } } ```
