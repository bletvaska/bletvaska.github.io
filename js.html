## Strings ```javascript // strings can be defined as "string" let name = "juraj"; // or as 'string' let surname = 'Holmes'; let age = 33; // strings can be concatenated with '+' operator console.log(name + ' ' + surname + " is " + age + ' years old'); // but you can use template literals instead console.log(`${name} ${surname} is ${age} years old`); ```
## Arrays ```javascript // this is array let numbers = [23, 42, 93, 34, 45]; let names = ['john', 'alice', 'lucy', 'tom']; // working with arrays numbers.push(99); // traverse array traditional way for(let i = 0; i < numbers.length; i++){ console.log(numbers[i]); } // or you can use "foreach" way for(let entry of numbers){ console.log(entry); } ```
### Function Declaration ```javascript function add(num1, num2){ return num1 + num2; } ``` ### Function Expression ```javascript let add = function(num1, num2){ return num1 + num2; } ```
## Objects * An **object** is a collection of **properties**. * A **property** is an association between a **name** (or key) and a **value**. * Multiple ways to create an `Object`: * object constructor: `obj = new Object()` * object literal: `obj = {}` * inbuild method: `obj = Object.create()` * constructor function: `obj = new Person()`
## Object Literal ```javascript let person = { name: 'Sherlock', surname: 'Holmes', getName: function(){ return `${this.name} ${this.surname}`; } }; // usage console.log(person.name); person['surname'] = 'holmes'; ```
## Constructor Function ```javascript // definition function Person(name, surname, sex = 'M'){ this.name = name; this.surname = surname; this.sex = sex; this.getName = function(){ return `${this.name} ${this.surname}`; }; } // usage let person = new Person('Sherlock', 'Holmes'); ```
## Prototype ```javascript function Person(name, surname, sex = 'M'){ this.name = name; this.surname = surname; this.sex = sex; } Person.prototype.getName = function(){ return `${this.name} ${this.surname}`; }; ```

## Quizz Create selector to 1. get all ```h1``` elements 2. ...
## Selectors Tips * never use universal selector ```'*'``` * use identifiers * use enough-specific selectors (but not too much) * use methods
## ```.hide()``` and ```.show()``` * [```.hide()```](http://api.jquery.com/hide/) - hide the matched elements * [```.show()```](http://api.jquery.com/show/) - display the matched elements
## ```.append()``` and ```.prepend()``` * [```.append()```](http://api.jquery.com/append/) - insert content, specified by the parameter, to the end of each element in the set of matched elements * [```.prepend()```](http://api.jquery.com/prepend/) - insert content, specified by the parameter, to the beginning of each element in the set of matched elements ```javascript // example usage $('h2').append(' STOP'); $('h2').prepend('START '); $('ul').append('<li>new entry</li>'); $('ul').prepend('<li>new entry</li>'); ```
## ```.html()``` and ```.text()``` * [```.html()```](http://api.jquery.com/html/) - get the HTML contents of the first element in the set of matched elements or set the HTML contents of every matched element * [```.text()```](http://api.jquery.com/text/) - get the combined text contents of each element in the set of matched elements, including their descendants, or set the text contents of the matched elements ```javascript // example usage $('section#html-text').html(); $('section#html-text').text(); $('section#html-text h2').html('<i>Heading</i>'); $('section#html-text h2').text('<i>Heading</i>'); ```
## ```.detach()``` * [```.detach()```](http://api.jquery.com/detach/) - remove the set of matched elements from the DOM ```javascript // example usage $('li').detach(); $('section#detach').detach(); ```
