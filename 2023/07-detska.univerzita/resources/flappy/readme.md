# Flap.py

## Steps

1. vytvorenie okna
    - sirka, vyska, titulok

2. nakreslenie flappyho
    - akter
    - suradnicovy system

3. herna slucka
    - priklad so super mariom

4. rozpohybovanie flappyho volnym padom
    - zacat len s flappy.y += 1

5. zahada - zostava po nom hnedy pas
    - ako sa robia filmy
    - spritesheet
    - animacia
    - zmazeme vzdy obrazovku
    - premazeme ju obrazkom

6. volny pad
    - flappy.vy += GRAVITY
    - flappy.y += flappy.vy

7. smrt, ked vybehne z obrazovky
    - handles

8. stlacenie klavesy
    - kick
    - osetrenie stropu

9. pipes na pozadi
