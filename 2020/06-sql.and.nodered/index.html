## [![SQLite](images/logo-sqlite.png)](https://www.sqlite.org/index.html)
## SQLite Features I. * Transactions are atomic, consistent, isolated, and durable (ACID) even after system crashes and power failures. * Zero-configuration. * Full-featured SQL implementation with advanced capabilities. * A DB is stored in a single cross-platform disk file. * Supports terabyte-sized databases and gigabyte-sized strings and blobs.
## SQLite Features II. * Simple, easy to use API. * Fast * Written in ANSI-C * Well-commented source code with 100% branch test coverage. * Self-contained: no external dependencies. * Cross-platform * Comes with a standalone CLI client.
## Suggested Uses * Database For The Internet Of Things * Application File Format * Website Database * Stand-in For An Enterprise RDBMS (for demonstration purposes or for testing)
